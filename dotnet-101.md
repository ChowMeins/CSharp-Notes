# .NET Notes
## What is .NET?
.NET is a software development platform/framework created by Microsoft.

Think of it as a toolbox and runtime environment for building applications.

It provides:

1. Runtime (CLR) – Common Language Runtime
- Executes your programs
- Handles memory management, garbage collection, exceptions, security
2. Base Class Libraries (BCL)
- Prebuilt code for common tasks:
    - Strings, collections, file I/O, networking, threading

3. Frameworks/SDKs built on top:
- ASP.NET → Web applications & APIs
- .NET MAUI → Cross-platform mobile/desktop apps
- WinForms / WPF → Desktop apps
- Entity Framework → Database access

4. Tools
- Command-line compiler (dotnet)
- Package manager (NuGet)
- IDE support (VS, VS Code)

## 💡 Analogy:
- .NET = the engine & toolbox for your programs
- C# = the language you use to tell the engine what to do

### Example answer if asked in an interview:

“C# is a modern, object-oriented programming language, and .NET is the development platform that provides the runtime, libraries, and tools to execute C# programs. The C# compiler (Roslyn), which is provided as part of the .NET SDK, generates intermediate language (IL) code that then runs on the .NET runtime, making C# applications cross-platform and fully managed.”
